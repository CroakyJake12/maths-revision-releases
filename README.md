# Maths Revision

A portable Windows revision app for Edexcel A-level Maths, including topic
videos, revision resources, progress tracking, study tools, and a built-in
practice bank.

## Download

[Download the latest portable version](https://github.com/CroakyJake12/maths-revision-releases/releases/latest/download/Maths.Revision.Portable.zip)

No installer or .NET runtime is required. Extract the ZIP and open
`Maths Revision.exe`.

## Automatic updates

Maths Revision checks this repository's `update.json` whenever it starts. New
versions are downloaded over HTTPS and verified with SHA-256 before the
executable is replaced and reopened.

> Windows may show a SmartScreen warning because the executable is not
> currently code-signed.
